# 📚 迭代信息 - Novel Reader

## 项目起源

本项目从漫画阅读器（MoriSora Manga Reader）改造而来。原始项目基于 Tauri 2 + React + Rust，专注于漫画/图片阅读。经过两个主要版本的迭代，改造为小说/文本阅读器。

---

## v0.1.0 - 漫画阅读器（原始版本）

### 功能概述
基于 Tauri 2 的桌面漫画阅读器，支持 CBZ/CBR/PDF 格式的漫画文件阅读。

### 核心功能
- CBZ/CBR 压缩包解析（zip/unrar）
- PDF 图片渲染（pdfjs-dist）
- 图片缩放、拖拽平移
- 双页/单页阅读模式
- 自动翻页
- 封面缩略图生成与缓存
- 文件夹树导航
- 收藏和标签系统
- 拖拽导入文件
- 右键菜单操作

### 技术栈
| 层级 | 技术 |
|------|------|
| 桌面框架 | Tauri 2 |
| 前端 | React 18 + TypeScript + TailwindCSS |
| 状态管理 | Zustand |
| 数据库 | SQLite (rusqlite) |
| PDF 渲染 | pdfjs-dist (前端) |
| 压缩包 | zip / unrar (Rust) |
| 图片处理 | image crate (Rust) |

### Rust 模块
- `archive_parser.rs` — ZIP/CBR 压缩包解析
- `cover_cache.rs` — 封面缩略图缓存
- `database.rs` — 数据库操作（manga_books 表）
- `events.rs` — 事件系统
- `file_operations.rs` — 文件操作
- `folder_manager.rs` — 文件夹管理
- `library_scanner.rs` — 目录扫描（cbz/cbr/pdf）
- `settings.rs` — 设置管理
- `sort_utils.rs` — 自然排序

### 数据库表
- `manga_books` — 漫画元数据（path, title, source_type, page_count, last_opened）
- `favorites` — 收藏
- `tags` / `book_tags` — 标签系统

## v0.2.0 - 小说阅读器（当前版本）

### 改造概述

从漫画阅读器全面改造为小说阅读器，保留约 60% 的架构代码，核心渲染引擎和文件解析系统完全重写。

### 新增功能

#### 1. 文本文件解析系统（3 个新模块）

| 模块 | 文件 | 说明 |
|------|------|------|
| PDF 文本提取 | `pdf_text_extractor.rs` | 使用 lopdf 提取 PDF 页面文本，支持目录大纲提取 |
| TXT 解析 | `txt_parser.rs` | 纯文本解析，chardetng 自动编码检测（UTF-8/GBK/GB2312/Big5），智能章节分割（匹配"第X章"、"Chapter X"等模式） |
| Markdown 解析 | `md_parser.rs` | pulldown-cmark 转 HTML，支持表格/删除线/任务列表，从标题层级自动提取目录 |

#### 2. 全文搜索引擎

| 模块 | 文件 | 说明 |
|------|------|------|
| 搜索引擎 | `search_engine.rs` | 支持 PDF/TXT/MD 三种格式的全文搜索，大小写不敏感，返回匹配位置和上下文片段 |

新增 Tauri 命令：`search_in_pdf`、`search_in_txt`、`search_in_md`

#### 3. 阅读统计系统

| 模块 | 文件 | 说明 |
|------|------|------|
| 阅读统计 | `reading_stats.rs` | 记录阅读会话（开始/结束时间、时长、页数），提供总览统计（总会话数、总时长、平均时长、最长会话） |

新增 Tauri 命令：`start_reading_session`、`end_reading_session`、`get_reading_stats`、`get_recent_sessions`

#### 4. 阅读器完整功能

| 功能 | 说明 |
|------|------|
| 文本渲染引擎 | PDF 页面文本、TXT 纯文本、Markdown HTML 渲染，替代原图片渲染 |
| 分页/滚动双模式 | 分页模式按页翻阅，滚动模式连续阅读 |
| 排版设置 | 字号 12-36px（步进 2）、行距 1.2-3.0（步进 0.2） |
| 三主题切换 | 暗色（#1A1A1A）、亮色（#FFFFFF）、护眼（#F4ECD8），每种主题独立配置文字/代码/引用/链接颜色 |
| 章节导航侧栏 | TXT 智能章节分割 + MD 标题层级缩进，点击跳转 |
| 自动进度保存 | 打开时自动恢复上次阅读位置，翻页后 3 秒防抖自动保存，关闭窗口时保存 |
| 书签系统 | 添加/删除/跳转书签，书签面板管理 |
| 高亮标注 | 选中文本浮动工具栏，4 色高亮（#CBE93A/#3AE9C8/#E93AC8/#E9C83A），添加笔记 |
| 全文搜索 | Ctrl+F 搜索框，前后跳转匹配项，当前匹配/其他匹配区分高亮 |
| 阅读统计 | 底部栏实时计时器，会话追踪，悬停查看总统计 |
| 键盘导航 | 方向键/PgUp/PgDown/Home/End/Ctrl+F/Esc |
| 滚轮分页 | 300ms 节流 |

#### 5. 书库界面改造

| 改动 | 说明 |
|------|------|
| 封面图 → 格式图标 | 移除封面缩略图，改为格式图标（PDF/TXT/MD）+ 格式徽章 |
| MangaItem → BookItem | 数据模型新增 author、wordCount、chapterCount 字段 |
| 搜索支持作者 | 搜索过滤同时匹配书名和作者 |
| 导航方式 | 双击书籍通过 hash 路由进入阅读器，替代原 WebviewWindow |

### 修改的功能

| 改动项 | 原实现 | 新实现 |
|--------|--------|--------|
| 项目名称 | manga-reader | novel-reader |
| 包名 | com.mangareader.app | com.novelreader.app |
| 支持格式 | cbz/cbr/pdf | pdf/txt/md |
| 渲染引擎 | pdfjs-dist 图片渲染 | lopdf 文本提取 + DOM 文本渲染 |
| 阅读模式 | 双页/单页图片 | 分页/滚动文本 |
| 进度保存 | 图片索引 | 章节 + 页码 + 偏移量 |
| 封面系统 | 压缩包首图提取 | 格式图标 + 默认封面 |
| 数据库 | manga_books 表 | book_metadata 表 + chapters/bookmarks/highlights/reading_sessions 表 |
| 编码处理 | 无 | chardetng 自动检测 + encoding_rs 转换 |
| 章节导航 | 无 | TXT 智能分割 + MD 标题层级 + PDF 大纲 |

### 废弃的功能

| 废弃模块 | 原因 |
|----------|------|
| archive_parser.rs（ZIP/CBR 解析） | 漫画专用压缩格式，小说不支持 |
| 图片渲染逻辑（pdfjs-dist） | 小说不需要图片渲染 |
| 双页阅读模式 | 不适合文字阅读 |
| 图片缩放/拖拽平移 | 改为字体大小调整 |
| 封面缩略图生成 | 改为格式图标 |
| 自动翻页 | 改为手动翻页 + 滚轮分页 |
| image crate 依赖 | 不再需要图片处理 |
| unrar crate 依赖 | 不再需要 CBR 解析 |
| zip crate 依赖 | 不再需要 CBZ 解析 |

### 数据库变更

#### 新增表

| 表名 | 用途 | 关键字段 |
|------|------|----------|
| chapters | 章节结构 | book_id, chapter_number, title, start_offset, end_offset, level |
| bookmarks | 书签 | book_id, chapter_id, offset, title |
| highlights | 高亮标注 | book_id, chapter_id, start_offset, end_offset, text_content, note, color |
| reading_sessions | 阅读会话 | book_id, start_time, end_time, duration_seconds, pages_read |

#### 扩展字段（book_metadata 表）

|----------|------|------|
| author | TEXT | 作者 |
| word_count | INTEGER | 字数 |
| chapter_count | INTEGER | 章节数 |
| encoding | TEXT | 文件编码 |

#### 阅读进度扩展（reading_progress 表）

| 新增字段 | 类型 | 说明 |
|----------|------|------|
| current_chapter | INTEGER | 当前章节 |
| chapter_offset | INTEGER | 章节内偏移 |
