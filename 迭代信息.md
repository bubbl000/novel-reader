# 迭代信息 - Novel Reader

## 项目起源

本项目从漫画阅读器（MoriSora Manga Reader）改造而来。原始项目基于 Tauri 2 + React + Rust，专注于漫画/图片阅读。经过多个版本的迭代，改造为小说/文本阅读器。

---

## v0.1.0 - 漫画阅读器（原始版本）

### 功能概述
基于 Tauri 2 的桌面漫画阅读器，支持 CBZ/CBR/PDF 格式的漫画文件阅读。

### 核心功能
- CBZ/CBR 压缩包解析（zip/unrar）
- PDF 图片渲染（pdfjs-dist）
- 图片缩放、拖拽平移
- 双页/单页阅读模式
- 自动翻页
- 封面缩略图生成与缓存
- 文件夹树导航
- 收藏和标签系统
- 拖拽导入文件
- 右键菜单操作

### 技术栈
| 层级 | 技术 |
|------|------|
| 桌面框架 | Tauri 2 |
| 前端 | React 18 + TypeScript + TailwindCSS |
| 状态管理 | Zustand |
| 数据库 | SQLite (rusqlite) |
| PDF 渲染 | pdfjs-dist (前端) |
| 压缩包 | zip / unrar (Rust) |
| 图片处理 | image crate (Rust) |

### Rust 模块
- `archive_parser.rs` — ZIP/CBR 压缩包解析
- `cover_cache.rs` — 封面缩略图缓存
- `database.rs` — 数据库操作（manga_books 表）
- `events.rs` — 事件系统
- `file_operations.rs` — 文件操作
- `folder_manager.rs` — 文件夹管理
- `library_scanner.rs` — 目录扫描（cbz/cbr/pdf）
- `settings.rs` — 设置管理
- `sort_utils.rs` — 自然排序

### 数据库表
- `manga_books` — 漫画元数据（path, title, source_type, page_count, last_opened）
- `favorites` — 收藏
- `tags` / `book_tags` — 标签系统

## v0.2.0 - 小说阅读器（初始改造版本）

### 改造概述

从漫画阅读器全面改造为小说阅读器，保留约 60% 的架构代码，核心渲染引擎和文件解析系统完全重写。

### 新增功能

#### 1. 文本文件解析系统（3 个新模块）

| 模块 | 文件 | 说明 |
|------|------|------|
| PDF 文本提取 | `pdf_text_extractor.rs` | 使用 lopdf 提取 PDF 页面文本，支持目录大纲提取 |
| TXT 解析 | `txt_parser.rs` | 纯文本解析，chardetng 自动编码检测（UTF-8/GBK/GB2312/Big5），智能章节分割（匹配"第X章"、"Chapter X"等模式） |
| Markdown 解析 | `md_parser.rs` | pulldown-cmark 转 HTML，支持表格/删除线/任务列表，从标题层级自动提取目录 |

#### 2. 全文搜索引擎

| 模块 | 文件 | 说明 |
|------|------|------|
| 搜索引擎 | `search_engine.rs` | 支持 PDF/TXT/MD 三种格式的全文搜索，大小写不敏感，返回匹配位置和上下文片段 |

#### 3. 阅读统计系统

| 模块 | 文件 | 说明 |
|------|------|------|
| 阅读统计 | `reading_stats.rs` | 记录阅读会话（开始/结束时间、时长、页数），提供总览统计 |

#### 4. 阅读器完整功能

| 功能 | 说明 |
|------|------|
| 文本渲染引擎 | PDF 页面文本、TXT 纯文本、Markdown HTML 渲染 |
| 分页/滚动双模式 | 分页模式按页翻阅，滚动模式连续阅读 |
| 排版设置 | 字号 12-36px、行距 1.2-3.0 |
| 三主题切换 | 暗色、亮色、护眼 |
| 章节导航侧栏 | TXT 智能章节分割 + MD 标题层级缩进 |
| 自动进度保存 | 打开时恢复位置，翻页后 3 秒防抖保存 |
| 书签系统 | 添加/删除/跳转书签 |
| 高亮标注 | 选中文本浮动工具栏，4 色高亮，添加笔记 |
| 全文搜索 | Ctrl+F 搜索框 |
| 阅读统计 | 底部栏实时计时器 |

### 修改的功能

| 改动项 | 原实现 | 新实现 |
|--------|--------|--------|
| 项目名称 | manga-reader | novel-reader |
| 支持格式 | cbz/cbr/pdf | pdf/txt/md |
| 渲染引擎 | pdfjs-dist 图片渲染 | lopdf 文本提取 + DOM 文本渲染 |
| 阅读模式 | 双页/单页图片 | 分页/滚动文本 |
| 进度保存 | 图片索引 | 章节 + 页码 + 偏移量 |
| 封面系统 | 压缩包首图提取 | 格式图标 + 默认封面 |
| 数据库 | manga_books 表 | book_metadata 表 + chapters/bookmarks/highlights/reading_sessions 表 |

### 废弃的功能

| 废弃模块 | 原因 |
|----------|------|
| archive_parser.rs（ZIP/CBR 解析） | 漫画专用压缩格式，小说不支持 |
| 图片渲染逻辑（pdfjs-dist） | 小说不需要图片渲染 |
| 图片缩放/拖拽平移 | 改为字体大小调整 |
| 封面缩略图生成 | 改为格式图标 |
| 自动翻页 | 改为手动翻页 + 滚轮分页 |
| image / unrar / zip crate | 不再需要 |

## v0.3.0 - 小说阅读器（功能优化版本）

### 改造概述

在 v0.2.0 基础上，重点优化阅读体验、UI 交互和国际化支持。

### 新增功能

#### 1. 国际化系统（i18n）

| 文件 | 说明 |
|------|------|
| `src/i18n/translations.ts` | 中英文翻译字符串（library/reader/settings 三个模块，150+ 翻译键） |
| `src/i18n/useTranslation.ts` | `useTranslation` hook + `t()` 全局翻译函数 |

- 语言状态存储在 Zustand store，全局共享
- 语言偏好持久化到 `localStorage`
- 设置对话框提供简体中文 / English 切换
- 所有 UI 文本（书库、阅读器、设置）均已国际化

#### 2. 三种阅读模式

| 模式 | 中文 | 英文 | 说明 |
|------|------|------|------|
| 滚动 | 滚动 | Scroll | 连续滚动阅读，按章节懒加载 |
| 单页 | 单页 | Single | 一页一页翻阅（原分页模式） |
| 双页 | 双页 | Spread | 左右两页并排，类似实体书 |

工具栏从单按钮切换改为三按钮组，当前模式高亮显示。

#### 3. 滚动模式章节懒加载

- 初始化只加载第一章内容
- 滚动到 80% 时自动加载下一章
- 缓存上限 3 个章节，超出时移除最早的
- TXT：根据 `start_offset` / `end_offset` 切割文本
- PDF：按页码范围分配章节
- MD：保持全文加载（文件通常较小）

#### 4. 侧栏可调宽度

- 目录/书签/高亮三个侧栏均支持拖拽调整宽度
- 左侧边缘拖拽手柄，悬停时显示绿色高亮
- 宽度范围：200px ~ 500px，默认 280px

### 修改的功能

| 改动项 | 原实现 | 新实现 |
|--------|--------|--------|
| 阅读模式 | 分页/滚动双模式 | 滚动/单页/双页三模式 |
| 工具栏按钮 | BM/HL 文字按钮 | 书签/笔记图标按钮（RxBookmark/RxPencil1） |
| 添加书签 | 工具栏按钮 | 移入书签侧栏顶部，文本按钮 |
| 导航按钮定位 | `fixed` 全局固定 | `absolute` 相对内容区域，侧栏开启时自动调整 |
| 侧栏宽度 | 固定 260px/300px | 可拖拽调整 200-500px |
| 章节跳转 | 跳转后自动关闭侧栏 | 跳转后保持侧栏打开 |
| 书签重复 | 可重复添加同一页 | 检测重复，提示"此页已有书签" |
| 语言设置 | 组件内部 useState | Zustand 全局状态 + localStorage 持久化 |
| 窗口关闭 | `close()` 触发权限错误 | `destroy()` + `close()` fallback |
| 书籍详情面板 | 有关闭按钮 | 移除关闭按钮 |

### 移除的功能

| 移除项 | 原因 |
|--------|------|
| 阅读器搜索功能 | 简化阅读器界面，书库搜索保留 |
| 阅读计时统计 | 简化底部状态栏 |
| 工具栏关闭按钮 | 使用系统标题栏关闭 |

### Bug 修复

| 问题 | 修复 |
|------|------|
| 默认语言显示英文 | 创建 i18n 系统，默认中文 |
| 语言切换不生效 | 语言状态提升到 Zustand 全局共享 |
| 阅读器窗口无法关闭 | 添加 `core:window:allow-destroy` 权限 + `destroy()` 方法 |
| 滚动模式卡顿 | 滚动事件 100ms 节流，无高亮时跳过 `renderAnnotatedText` |
| 添加书库路径后扫描无效 | 修复 `handleSelectFolder` 异步 state 竞态，改为直接使用 `open()` 返回值调用 `addLibraryPath` |
